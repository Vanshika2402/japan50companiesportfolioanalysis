# Japanese 50 Companies Quantitative Portfolio Analysis

This project analyzes 50 publicly listed Japanese companies using historical stock price data.

## Features
- Downloads historical stock data from Yahoo Finance
- Calculates daily returns
- Computes risk-return metrics
- Calculates Sharpe ratio, beta, downside risk, drawdown, and VaR
- Builds equal-weighted, minimum variance, and maximum Sharpe portfolios
- Generates charts and CSV outputs

## Technologies Used
- Python
- pandas
- numpy
- yfinance
- matplotlib
- seaborn
- scipy
